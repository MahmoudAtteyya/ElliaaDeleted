## Installation

1. Download the APK
2. Enable "Install from unknown sources" in Android settings
3. Open and install
4. Grant notification access permission
5. Done! Deleted messages will appear in the app

## How It Works

ElliaaDeleted uses Android's Notification Listener Service to capture incoming messages. When a message is deleted from the original app (WhatsApp, Telegram, Signal), ElliaaDeleted retains a copy because it received the notification before the deletion.

**Important**: Your messages stay on your device. We don't collect, store, or transmit any data.
